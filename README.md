# Jam Desi Bites Restaurant Management System

This project is a C++ based Restaurant Management System that we developed to manage basic restaurant operations in a simple and organized way. It was created by Abdullah Ahmad, Muhammad Aslam, and myself under the guidance of Sir Asim Arshad.

## About the Project

The system helps manage day-to-day restaurant tasks like taking orders, handling inventory, managing employees, table reservations, customer feedback, and generating reports. The main purpose of this project was to apply our C++ concepts in a real-world type system using file handling and modular programming.

## Features

- Login system for users  
- Customer and employee management  
- Order taking and processing  
- Inventory tracking  
- Table reservation system  
- Delivery management  
- Customer feedback handling  
- Basic financial and revenue reports  

## Technical Work

We used C++ for the whole project. File handling was used to store data like orders, inventory, and payments so the data stays saved even after closing the program. The project is divided into functions to keep everything organized, like order handling and inventory management. We also added input validation to avoid wrong entries and basic error handling for stability.

## Challenges We Faced

We had some issues in updating inventory correctly and managing calculations for revenue. Also, input validation was a bit tricky at some points. We solved these problems by improving our logic and using proper file handling techniques.

- Problem solving  
- Data validation  
- Modular programming

This project helped us understand how a real system works and improved our programming and teamwork skills.
